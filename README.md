# 🎮 Hangman Game

A simple **Hangman Game** written in **C** using **raylib**.  
This project was built as a learning exercise in low-level game development.

---

## 🚀 Getting Started

### Requirements
- C compiler (GCC / Clang / MSVC)
- **raylib** installed on your system


- Official site raylib.com

---

## 🔧 Build & Run

### (WINDOWS) Using Notepad++ for Raylib (Recomended)
A "Notepad++ for raylib" shortcut is created on desktop on installation and when the program is first opened, 
it opens raylib.h header file, for API reference, and core_basic_window.c example. 

Open hangman.c file on Notepad++ for raylib and pressing F6 should open NppExec script system to compile and run the game.
